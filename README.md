# NextStep 👟

A responsive e-commerce shoe store built with Next.js, TypeScript, and Tailwind CSS v4. Features product browsing, category filtering, cart management, and a custom navy & gold design system.

---

## Tech Stack

Next.js 15 · TypeScript · Tailwind CSS v4 · Redux Toolkit · Framer Motion · Lucide React · React Toastify · Web3Forms

---

## Features

- 🛍️ Product browsing with search & filter (gender, category)
- 👟 Dynamic product detail pages
- 🛒 Cart drawer with add, remove & quantity controls
- 📬 Working contact form via Web3Forms
- ✨ Page load animations with Framer Motion
- 📱 Fully responsive design

---

## Getting Started

```bash
git clone https://github.com/wajahat-kamal/NextStep-Nextjs.git
cd nextstep
npm install
npm run dev
```
